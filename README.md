
# Todo Application

This application is used to create the todo lists. It uses MongoDB Atlas for storing the data.


## Run Locally

Clone the project

```bash
  git clone https://github.com/erabhinavrana/my-todolist-app.git
```

Go to the project directory

```bash
  cd my-todolist-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon app.js
```
